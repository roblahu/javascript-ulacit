Instrucciones del Hackaton

1. Usando el AJAX que se vio como ejemplo, consultar la lista de usuarios disponible

2. Una vez se alla leido el AJAX se debe mostar una tabla con la lista de todos los usuarios, incluida la imagen del usuario

3. Utilizando JQUERY, se debe poder eliminar de la tabla un usuario (Solo eliminar de la tabla local creada con la respuesta)

4. Usando Jquery, poder agregarle una descripción a cada usuario, puede quedar en un Array temporal dicha descripción
